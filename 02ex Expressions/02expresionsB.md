# expressions

## numeric expressions

    - operators ( +, -, *, /, **, % )
    - + addition
    - - subtraction
    - * multiplication
    -  / division
    -  ** power
    -  % remainder

## operation precedence

    - which operator take precedence?
        1. Parenthesis ()
        2. power **
        3. multiplication *
        4. additon +
        5. left to right
